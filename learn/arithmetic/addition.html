Addition

Simple (Grades 1–5) | Higher-Level (Grades 6+)

Concept

Addition is the process of combining two or more numbers to find the total or sum. It is one of the most fundamental operations in mathematics and is used in everyday life.

Step-by-Step Instructions

  1. Identify the numbers you want to add together.
  2. Start with the first number.
  3. Add the second number to it, counting up if necessary.
  4. For larger numbers, write the numbers in columns aligned by place value (ones, tens, hundreds).
  5. Add each column starting from the rightmost (ones), carrying over any extra value to the next column.

Hands-On Techniques

Here are some ways to make addition easier and more intuitive:

  • Use your fingers for small numbers (1–10).
  • Draw dots, tally marks, or counters for visual addition.
  • Write numbers in columns for multi-digit addition and carry over as needed.
  • Break numbers into parts (place value) for easier mental addition. Example: 47 + 36 = 40 + 30 + 7 + 6 = 70 + 13 = 83.

Worked Examples

Example 1 (Simple): 3 + 5 = 8
Count up: Start at 3, then count 4,5,6,7,8 → Total = 8

Example 2 (Using Fingers): 6 + 4
Hold up 6 fingers on one hand, then 4 on the other. Count all fingers together = 10.

Example 3 (Column Addition): 47 + 36

          47
        + 36
        ----
          83
Add ones: 7 + 6 = 13 → write 3, carry over 1.
Add tens: 4 + 3 = 7, plus the carried over 1 = 8 → Total = 83

Example 4 (Breaking Apart Numbers): 57 + 28
Break into tens and ones: (50 + 20) + (7 + 8) = 70 + 15 = 85

Practice
